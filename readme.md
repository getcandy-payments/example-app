# GetCandy Payments Example App

Laravel App demonstrating how to use GetCandy Payments.

## Currently supported

- [ ] Offline
- [ ] Stripe
- [ ] PayPal
- [ ] SagePay
- [ ] Klarna Checkout
- [ ] Mollie
- [ ] Braintree Payments

## Documentation

Docs can be viewed by running the app and browsing to `/docs`.
